Deployment boundary

这个站点是项目门户，不是发布系统。

它应该组织公开说明、想法、提案和策展式预览；不应该维护分支快照、生成构建、 私密资料或访问控制。

适合放在这里

公开说明、路线图、概念条目、预览摘要、外部链接和可讨论的提案。

不适合放在这里

实现代码、完整 demo 构建、分支快照、敏感资料和需要真实权限控制的内容。
