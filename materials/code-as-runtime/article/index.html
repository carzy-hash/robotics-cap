Code as Runtime · Architecture note

Agent 通过代码，
维护机器人运行时。

Code as Runtime 把模型输出从一次性的 action 或 program，变成可持续执行、观察和改写的控制层。每轮决策中，主 Agent 可以运行代码构建 Action Chunk，也可以安装按频率或事件触发的 handler；两类代码跨越决策周期，共同维护机器人正在消费的 Action Queue。
